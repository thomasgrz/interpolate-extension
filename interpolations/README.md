# Interpolations

Interpolation configurations allow you configure the "interpolate" web extension.

To get started find the interpolation you'd like to import into interpolate.

1. [headers interpolations](./headers/)
2. [redirect interpolations](./redirects/)
3. [mock API interpolations](./mock-apis)
4. [user script interpolations](./scripts/)

In the interpolate web extension, click "Create interpolation".

Then select "Import" in the options modal that appears.

Copy and paste the JSON blob for any interpolation and paste it into the "Import" text area.

A preview of the interpolation card should be displayed.

Click "Save"

Have fun!
